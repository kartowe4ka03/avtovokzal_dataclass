Привет! Это мой первый проект на тему "Составные данные". 
В нем я создал очень простую программу, которая основана на работе с Dataclass

Данная программа может: 
1)Создавать файлы
2)Добавлять в JSON-файл данные
3)Удалять из JSON-файла данные
4)Выводить данные из JSON-файла в консоль
5)Искать по трем критериям: фамилия, перевозчик, город прибытия

Программа запускается через консоль при помощи файла main.py

-Чтобы начать работу с файлом, введите в консоли py main.py -h
Данная команда покажет вседоступные команды в данной программе.
Они строятся про принципу py main.py 'команда' data.json
Вместо data.json может быть любой ваш JSON-файл.
В дальнейшем планирую оптимизировать удаление из файла, 
чтобы не приходилось полностью вводить данные о клиенте
